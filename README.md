# BS Oils Website

A React website with CSS Modules, a home page, and product subpages for oil offerings.

🌐 **Live Site:** https://sacredalchemy.github.io

📦 **GitHub Repository:** https://github.com/sacredalchemy/sacredalchemy.github.io

## Features

- 🏠 Modern responsive design
- 📱 Mobile-friendly interface
- 🌓 Dark/Light theme toggle
- 📅 Booking system with date/time picker
- 💬 WhatsApp integration
- 🎴 Tarot readings showcase
- 🕯️ Candle spells catalog
- 🧴 Oil products & services
- ✨ Spiritual guidance services

## Quick Setup

1. `npm install`
2. `npm start`
3. Visit http://localhost:3000

## Build & Deploy

```bash
npm run build
```

The production build is automatically deployed to GitHub Pages: https://sacredalchemy.github.io

## Structure

- `src/components` — reusable UI pieces with module CSS
- `src/pages` — routed page views
- `src/data` — oil product content
- `build/` — production build (deployed to GitHub Pages)
